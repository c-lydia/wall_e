micro-ROS Workspace for ESP32 Development
==========================================

Welcome to the micro-ROS workspace documentation. This project integrates ESP32 firmware development with ROS 2 Humble for embedded robotics applications.

Overview
========

This workspace provides a complete development environment for:

- ESP32 firmware development with FreeRTOS
- micro-ROS integration for embedded systems
- ROS 2 Humble host environment
- Docker-based containerization for reproducible builds

Key Features
============

- **Docker Compose**: Multi-container development setup
- **ROS 2 Humble**: Latest ROS 2 distribution
- **micro-ROS Agent**: UDP bridge for ESP32 communication
- **Sphinx Documentation**: Complete project documentation
- **VS Code Integration**: C++ IntelliSense and debugging support
